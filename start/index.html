The honest boundary

The census cohort is a disclosed closed experiment (our own agents, not external adoption); TOKEN is an internal accounting unit with no monetary value. What's real and checkable: every record is EIP-191 / Ed25519 signed and Merkle-sealed. We sign facts, not judgments.
